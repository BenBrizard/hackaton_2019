pour ajouter existing project :

va dans le dossier, ouvre git bash, fait 
git add .
git commit -m "blabla"
git remote add origin git@github.com:BenBrizard/Digital-clock.git // cette ligne défini le nom du projet sur github
git remote -v //pour vérifier que tout est beau
git push origin master


ensuite si tu veux re modifier des shits

1-git pull origin master (vérifie que t'es a jour avant de push tes modif)
2- git add .
3- git commit -m "changed spi_master_TB"
4- git push origin master

En tout temps, faire "git status" t'indique la solution
